ROMs

This page contains ROMs from Android 10 (excluding crDroid 6.27, which is based on Android 11 firmware.), and up to the current latest Android version, which is 15. Android 9 is not included, since most users are using, or have updated to Stock ROM Android 11 version, which triggers rollback projection. Means, you cannot install an older Android version than 11. If you do, your phone will get bricked.

Warning: you CANNOT install ANY of these ROMs IF you are using MIUI, or have converted your phone to MIUI/CC9e (or whatever the hell it is called, I don't know). To flash one of these ROMs, you MUST flash stock ROM, preferably (?) the fastboot version. You can check the guide here.

Active ROMs

ROMs under this category are actively, or just regularly maintained. A ROM will be under this category as long as the maintainer keeps supporting it. If they discontinue it, it will be moved to the Archived category (unless the download links are down. In this case, the ROM is removed from the website completely.)

"FAQ"

  • Which ROM should i use?

    • It all depends on your needs. Do you want customization? Then pick EvolutionX, for example. Do you want something lightweight, something minimal? Pick LineageOS (I'm currently using LineageOS 22.1, by the way.)

  • How do i flash XYZ ROM?

    • Please check the support group for the flashing instructions. All ROMs have a telegram support group linked. Join, and simply ask for flashing instructions.

  • Which ROM/kernel is the best??!!!!?!

    • Stop asking for the best ROMs/Kernels (for battery, for performance or any other criteria), it doesn't exist. If there was one true best ROM/Kernel, do you think there would be other ROMs or Kernels? There's only subjectively best and that's different for everyone. Everyone has different usage patterns & preferences, so when asking for the best, you get the best for them, not the best for you. The only way you'll find the best one for you is by trying them yourself. They're all free, anyway. BEST IS SUBJECTIVE.
