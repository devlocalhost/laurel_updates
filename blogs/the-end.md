# The end
If you haven't noticed, the community is... kind of dead. The groups are not that active anymore, plus development has been slow too. Same thing for the website (this website). The last update (before this one) was on September 19, 2025. That's a lot of months of inactivity. I also did not have motivation to update and/or work on the website anymore, so, I'm deciding to archive the website.

## What does this mean for the website?
The website will not be taken down, but it won't receive any updates. No updates about the device (ROM, Kernel, etc) will be released.
